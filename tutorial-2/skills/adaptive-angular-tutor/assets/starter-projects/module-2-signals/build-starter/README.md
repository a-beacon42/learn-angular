# Module 2: Signals, input(), output(), derived state - Build Starter

This folder is an Angular standalone starter for the **Module 2: Signals, input(), output(), derived state** module.

## Variant Goal
Implement the feature from a near-blank baseline.

## Focus
Signal modeling, parent-child contracts, and reactive data flow.

## Suggested Flow
1. Run the tasks in 'TASKS.md'.
2. Implement or fix code in 'src/app/exercise.component.ts' and 'src/app/exercise.component.html'.
3. Validate behavior with quick manual checks or tests.
