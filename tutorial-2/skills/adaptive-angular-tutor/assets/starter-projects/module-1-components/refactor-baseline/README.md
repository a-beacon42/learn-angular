# Module 1: Components, templates, control flow - Refactor Baseline

This folder is an Angular standalone starter for the **Module 1: Components, templates, control flow** module.

## Variant Goal
Improve readability/architecture/performance without changing behavior.

## Focus

## Suggested Flow
1. Run the tasks in 'TASKS.md'.
2. Implement or fix code in 'src/app/exercise.component.ts' and 'src/app/exercise.component.html'.
3. Validate behavior with quick manual checks or tests.
